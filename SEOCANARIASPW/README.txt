SEO CANARIAS - Sitio web demo
Archivos incluidos:
- index.html
- contacto.html
- precios.html
- testimonios.html
- styles.css
- scripts.js

Descripción:
Sitio sencillo inspirado en Hostaway/Holidu con popups de acceso rápido (esquinas superior izquierda), chat de WhatsApp (abajo a la izquierda) y páginas separadas por apartado. Está pensado para ser fácil de editar por el cliente: todo el CSS principal está en styles.css y se usa Tailwind CDN.

Instrucciones de uso:
1. Descomprime el ZIP.
2. Sirve la carpeta con un servidor estático (p. ej. Live Server en VSCode) o abre index.html directamente.
3. Personaliza textos, imágenes y contactos (teléfono en el enlace de WhatsApp).

Contacto del desarrollador: reemplaza contacto@seocanarias.com y el número de WhatsApp por los reales.
